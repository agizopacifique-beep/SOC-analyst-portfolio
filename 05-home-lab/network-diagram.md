                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    ## Data Flow Explanation
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    ### Attack Flow (Red arrows)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    Kali Linux generates all attack traffic in every lab. It sends:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **nmap** packets to enumerate open ports on Windows 10 (Lab 1)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Hydra** SMB login attempts against the Administrator account (Lab 2)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Metasploit** payload delivery — shell.exe served via HTTP, reverse TCP connection on port 4444 (Lab 3)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **psexec.py** remote execution over SMB — installs PSEXESVC service on Windows (Lab 4)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **FTP exfiltration** — receives sensitive.txt transferred from Windows (Lab 5)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    ### Endpoint Log Flow (Solid blue/yellow arrows)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    Both target VMs run the Wazuh Agent which continuously reads local logs and forwards them to Security Onion in real time:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Windows 10** → Sysmon events (EID 1, 3, 11, 13, 22) + Windows Security Events (EID 4624, 4625, 4648, 4688, 7045)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Ubuntu Server** → /var/log/auth.log + /var/log/syslog
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    ### Network Detection Flow (Dotted green arrow)
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    Security Onion passively monitors all network traffic on the Host-Only subnet:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Suricata** inspects every packet against its signature rule database and fires alerts for known attack patterns
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    - **Zeek** automatically generates structured log files (conn.log, dns.log, http.log, ftp.log) for every connection

                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    ## Why Host-Only Network?
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    A Host-Only network exists entirely inside VirtualBox. Unlike NAT or Bridged networking:
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Network Type | Internet Access | Home Router Sees Traffic | Attack Traffic Escapes |
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |---|---|---|---|
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | **Host-Only (this lab)** | No | No | No |
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | NAT | Yes | No | No |
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | Bridged | Yes | Yes | Possible |
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    Host-Only is the correct choice for a security lab because all attack traffic — nmap scans, Hydra brute force, Metasploit payloads, FTP exfiltration — stays completely inside the virtual network and never touches a real network device.
